# GitHub Pages Hosting - Complete Guide

## 🚀 QUICK DEPLOYMENT (5 MINUTES)

### Step 1: Create a GitHub Account
1. Go to: https://github.com/signup
2. Sign up with your email
3. Verify your email address

### Step 2: Create a New Repository

1. Go to: https://github.com/new
2. Fill in the form:
   - **Repository name:** `product-list`
   - **Description:** "Product Portal Website"
   - **Visibility:** Public (required for free GitHub Pages)
   - **Initialize with:** None (leave blank)
3. Click "Create repository"

### Step 3: Push Your Code to GitHub

Open Command Prompt/PowerShell in: `c:\Users\MY PC\Desktop\product list`

Run these commands ONE BY ONE:

```bash
# Initialize Git
git init

# Add all files
git add .

# Create first commit
git commit -m "Initial commit: Product Portal Website"

# Add remote (REPLACE yourusername with YOUR GitHub username)
git remote add origin https://github.com/yourusername/product-list.git

# Rename branch to main
git branch -M main

# Push to GitHub
git push -u origin main
```

### Step 4: Enable GitHub Pages

1. Go to your repository: https://github.com/yourusername/product-list
2. Click **Settings** (top menu)
3. Click **Pages** (left sidebar)
4. Under "Source":
   - Select branch: **main**
   - Select folder: **/ (root)**
5. Click "Save"
6. Wait 1-2 minutes for deployment

### Step 5: Get Your Live URL

Your website will be at:
```
https://yourusername.github.io/product-list
```

Example: If your username is "johndoe":
```
https://johndoe.github.io/product-list
```

## ❌ TROUBLESHOOTING

**Problem: "Repository not found" error**
- Solution: Make sure you're using correct GitHub username
- Solution: Check that repository is PUBLIC

**Problem: Page takes time to load**
- Solution: GitHub Pages takes 1-2 minutes to deploy
- Try refreshing after 2 minutes

**Problem: Still shows old version**
- Solution: Hard refresh browser (Ctrl+Shift+R)
- Solution: Clear browser cache

**Problem: 404 error on products.html**
- Solution: Make sure `products.html` is in root folder
- Solution: Force refresh browser
