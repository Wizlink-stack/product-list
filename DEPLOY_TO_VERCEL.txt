🚀 STEP-BY-STEP DEPLOYMENT GUIDE
=================================

STEP 1: Initialize Git Repository
----------------------------------
1. Open Command Prompt or PowerShell
2. Navigate to: c:\Users\MY PC\Desktop\product list
3. Run these commands:

   git init
   git add .
   git commit -m "Initial commit: Product Portal Website"


STEP 2: Create GitHub Account (If you don't have one)
-----------------------------------------------------
1. Go to: https://github.com/signup
2. Sign up with email and password
3. Verify your email


STEP 3: Create GitHub Repository
--------------------------------
1. Go to: https://github.com/new
2. Repository name: product-list (or any name)
3. Description: "Product Portal with Login"
4. Click "Create repository"
5. Follow the instructions to push your local repo


STEP 4: Deploy to Vercel
------------------------
1. Go to: https://vercel.com
2. Click "Sign Up" → "Continue with GitHub"
3. Connect your GitHub account
4. Authorize Vercel
5. Click "Import Project"
6. Select "product-list" repository
7. Click "Import"
8. Click "Deploy" (takes ~30 seconds)


✅ YOUR LIVE WEBSITE URL:
https://product-list.vercel.app

(Vercel will assign you a custom subdomain automatically)


🎯 OPTIONAL: ADD CUSTOM DOMAIN
==============================

If you want your own domain (www.productlist.com):

1. Go to: https://www.freenom.com
2. Register a FREE domain
3. In Vercel Project Settings:
   - Go to "Domains"
   - Add your custom domain
   - Copy the DNS records
4. Update DNS in freenom
5. Wait 5-15 minutes for DNS propagation
